# Changes

This document highlights the most important fixes, improvements, and features, that were introduced in Webiny `5.8.1`.

## Headless CMS

### Fixed some bugs
1. Sometimes rich-text field was not compressed before storing it to the DynamoDB / Elasticsearch.
2. In the UI cache broke due to non-existing GraphQL query cache key.

### Storage operations types
There are some changes in the typings for the storage operations which make it a bit simpler to use.
